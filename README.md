# Interest Hub — Economy Edition

한국어 경제 관심사 대시보드입니다.

## 실행

```powershell
cd dashboard
python server.py
```

브라우저에서 `http://127.0.0.1:8787`을 엽니다.

## 포함 기능

- 기본 국내·미국 관심 종목과 시가총액
- 종목명 검색 및 관심 목록 저장
- 종목·코인 클릭형 7일 차트
- 크립토 프로젝트 순위와 헤더 정렬
- 밈코인 Top 30 및 체인별 Top 20
- Robinhood 체인 포함 체인 필터
- 주식·세계정세·코인 뉴스
- CoinGecko, Yahoo Finance, Alternative.me, Google News RSS 연동
